## 安装
通过执行下列语句就可以完成安装

	go get -u github.com/chanxuehong/wechat/...

## 文档  
### [在线文档](http://godoc.org/github.com/chanxuehong/wechat)  
### 离线文档  
通过上面步骤下载下来后，可以在shell(windows 下面是 cmd) 里运行

	godoc -http=:8080
	
然后在浏览器里地址栏输入 

	http://localhost:8080/
	
即可查看文档
